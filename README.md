# Globo collection
App desenvolvido em swift IOS

- [link do projeto](https://globo-collection.netlify.app/stickes/stickes)

## Autores

- [@LarissaLanes](https://www.github.com/LarissaLanes)

## Stack utilizada

**Front-end:** Swift IOS

**Back-end:** Firebase

## Pods

- Firebase Auth
- Sweet alert

## Funcionalidades

**Implementadas:** 
- Resgate do sticker
- Login
- SignIn
- LogOut
- Redirecionamento para o quizz globo

**Não implementadas:** 
- Ranking
- Pontuação
